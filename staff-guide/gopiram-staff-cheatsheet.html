B · Staff2 (limited — sirf 3 tab)

Neeche 5 tab dikhenge: Catalog Orders My Tasks Send More

1. Login

  1. App kholo → Username daalo → PIN daalo → Sign In.

2. Customer ko saree bhejna (Send)

  1. Neeche Send tab dabao.
  2. Saree chuno — By Item ya upar search box se dhoondho.
  3. Select Recipient → customer ya group chuno (naam/number se search).
  4. Photo branding ke saath + caption apne aap ready → Send.

3. Rate / Stock check

  1. Home screen ke search box me design number ya naam type karo.
  2. Price + stock (in stock / out) turant dikh jaayega.

4. Catalog dekhna / add karna

  1. Catalog → Brand → Item → Design.
  2. Naya item/design add karne ke liye + button.

5. Naya Order / Inquiry

  1. Orders → New Order / Inquiry.
  2. Design search karo → Customer name, phone, quantity, notes bharo → save.
  3. Status badalne ke liye order pe tap → Update Status.

6. Mera kaam (Tasks)

  1. My Tasks → apna kaam dekho (Overdue / Due today badge).
  2. Ho jaaye to Complete Task (note likh sakte ho).

7. Contacts (More tab)

  1. More → Contacts → naya customer add / edit.

Neeche sirf 3 tab dikhenge: Rates My Tasks Orders

1. Login

  1. App kholo → Username + PIN → Sign In.

2. Rate / Stock dekhna (Rates)

  1. Rates tab kholo.
  2. Search box me design number / item / brand type karo.
  3. Price + stock dikh jaayega. (Yahan sirf dekhna hai — catalog browse nahi.)

3. Mera kaam (My Tasks)

  1. My Tasks → apne assign kiye gaye kaam dekho.
  2. Overdue / Due today badge dhyan se dekho.
  3. Ho jaaye to Complete Task dabao (note optional).

4. Orders

  1. Orders tab me sab orders dikhenge.
  2. Order pe tap → Update Status se status badlo.
  3. Note: Naya order banane ka button nahi hai — sirf dekhna + status.
Staff2 ka kaam short me:

Rate batao · apna task pura karo · order ka status update karo. Bas itna.

Rozana ka aasaan tareeka (sabke liye)

  • Subah: apna My Tasks ek baar dekh lo.
  • Customer pooche rate: WhatsApp me type nahi — app me search karo.
  • Naya saree aaya: photo app se bhejo (branding lag jaayegi).
  • Order mila: turant Orders me daal do.
Golden Rules — yaad rakhna
  • Photo hamesha app se bhejo, gallery se WhatsApp pe direct nahi — warna branding nahi lagegi.
  • Ek device pe hi login rahega (staff ke liye). Naye phone pe login karoge to purana apne aap logout.
  • Apna PIN kisi ko mat batao.
  • Rate ya stock ka doubt ho to app pe check karo — memory/andaaza nahi.
  • Kaam pura ho to task Complete zaroor karo, taaki malik ko pata rahe.
